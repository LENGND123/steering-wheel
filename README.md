# ⚙️ How It Works

1. Launch the application.
2. The camera initializes automatically.
3. Hold both hands straight for calibration.
4. MediaPipe detects both hands.
5. The steering angle is calculated.
6. Dashboard updates in real time.
7. Steering direction changes according to hand movement.

## 1️⃣ Clone Repository

```bash

```

```bash

```

